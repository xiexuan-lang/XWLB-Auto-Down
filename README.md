
# <center>自动新闻联播下载器</center>
***  
### 1.总论  
<u>*本应用用于自动下载每日新闻联播节目*</u>    
**本应用主要使用python语言和html相关解析技术和爬虫相关技术**   
### 2.主要特点
1.每日定时下载当天新闻联播,MP4文件   
2.自动下载最新一期的新闻联播  
3.使用Toml配置本地下载目录  
4.You-Get集成  
5.全平台兼容

### 3.实现方法
* 对新闻联播html文件解析

* 采取配置文件形式确定下载目录和性质
