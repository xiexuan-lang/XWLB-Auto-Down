
# XWLB Auto Down
# 自动新闻联播下载器

## 概述

本应用用于自动下载每日新闻联播节目

**本应用主要使用Python语言和HTML相关解析技术和爬虫相关技术**   

## 特点

- 自动下载最新一期的新闻联播
- 使用Toml配置本地下载目录  
- You-Get集成  
- 全平台兼容

## 原理
* 对新闻联播html文件解析
* 对you-get原生代码进行改进以更适应本应用
* 采取配置文件形式确定下载目录和性质
